# docs/ 目录说明

## 📂 开发文档目录

本目录包含面向开发者的技术文档。

## 文件说明

### `architecture.md`
**作用**：系统架构设计文档  
**内容**：
- 四层架构图解（输入层→评分层→风控层→决策层）
- 数据流图（用户输入→数据采集→多维度评分→综合决策→输出报告）
- 核心模块详解
- 评分体系表格
- 核心算法伪代码
- 动态权重调整对照表
- 信号冲突检测逻辑
- 扩展性设计指南
- 性能考虑
- 安全与合规说明

**适用人群**：
- 想了解系统架构的开发者
- 想要扩展功能的贡献者
- 想要深入理解模型逻辑的研究者

**阅读顺序建议**：
```
1. architecture.md（架构概览）
2. src/config.py（配置参数）
3. src/analyzer.py（核心逻辑）
```
