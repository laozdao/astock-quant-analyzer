# skill/ 目录说明

## 📂 SOLO Skill 文件目录

本目录包含可直接导入 SOLO 智能助手的 Skill 文件。

## 文件说明

### `SKILL.md`
**作用**：SOLO 智能助手技能定义文件  
**功能**：
- 定义 Skill 的元信息（名称、描述、触发词）
- 完整描述双引擎四层模型架构
- 详细的六步分析流程指南
- 各因子/维度的评分标准
- 动态权重调整规则
- 信号冲突处理规则
- 输出报告模板（十大章节）

**使用方式**：
1. 下载 `SKILL.md` 文件
2. 在 SOLO 中导入 Skill
3. 直接使用：`分析一下贵州茅台`

**适用场景**：
- 当用户提到"分析A股股票"、"个股量化分析"
- 当用户提到"股票投资评级"、"买卖时机判断"
- 当用户提到"量价分析"、"基本面选股"
- 当用户提到"多因子评分"或"量化模型分析"

**触发关键词**：
```
A股、股票分析、量化分析、投资评级、买卖时机、
量价分析、基本面、技术面、多因子、选股
```

**文件内容结构**：
```
1. 核心理念
2. 模型架构（四层结构）
3. 分析流程（六步法）
4. 评分标准详解
5. 输出格式模板
6. 核心原则
7. 免责声明
```
