# examples/ 目录说明

## 使用示例目录

本目录包含完整的使用示例和分析报告范本。

## 文件说明

### `sample_report.md`
**作用**：完整分析报告示例  
**案例**：贵州茅台（600519）

**报告结构（十大章节）**：
1. **基本信息** - 股价、市值、行业、概念、52周区间
2. **行业板块分析** - 景气度、估值、排名、龙头对比
3. **概念板块分析** - 各概念关联度、热度、持续性
4. **消息面分析** - 近3月重大消息梳理、评分、量价验证
5. **基本面评分** - 八因子评分表 + 亮点 + 风险点
6. **量价评分** - 六维度评分 + 健康/危险信号 + 阶段
7. **风险评估** - 四维度评分 + 预警级别 + 冲突检测
8. **综合决策** - 得分、评级、操作、框架适配
9. **操作建议** - 买卖时机、仓位、止损、目标价
10. **风险提示** - 综合风险点提醒

**用途**：
- 了解 Skill 输出的报告格式
- 参照格式分析其他股票
- 学习量化分析报告的写法

**使用方式**：
直接下载文件查看完整报告示例
